## инструкция по запуску

---

1. созадть директорию сборки

```
mkdir build
cd build
```

2. сгенерировать проект

```
cmake ..
```

3. собрать проект

```
make
```

4. запустить (в фоновом режиме)

```
./app <path> <interval_seconds> &
```

5. вывести содержимое файла (в домашнем каталоге)

```
cat .media_files
```
